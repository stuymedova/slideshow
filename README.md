# JS Slideshow

A simple slideshow implemented in Vanilla JavaScript as part of the [howtocode.club](https://howtocode.club) course

## Overview

```
.
├── assets
│   ├── imgs
│   │   └── …
│   ├── main.css
│   └── main.js
└── index.html
```

## Prerequisites

[Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/)

## Setup

Run in terminal:
```
npm start
```
